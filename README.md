This is a simple and easy-to-use web-based to-do list app made with HTML, CSS, and JavaScript. It helps people keep track of their tasks with features like categories, priorities, due dates, and more. The app works right in the browser and saves your tasks locally.

### Why We Made This

This project was created as part of our Computer Science and Engineering course at ARYA College of Engineering and IT. Our goals were to:

- Learn and show skills in basic web development (HTML, CSS, JavaScript)
- Build something useful for everyday task management
- Work as a team on a real project
- Make an app that anyone can use easily

It's a practical tool that also shows our learning in web development.

### Prerequisites

### Installation
1. Clone or download the project files
2. Open index.html in your web browser
3. Start managing your tasks!

## ðŸ“– How to Use

### Adding Tasks
- Fill in the task title and due date
- Add description, category, and priority if needed
- Click "Add Task"

### Managing Tasks
- Click the checkbox to mark tasks as done
- Use the edit button to change tasks
- Use the delete button to remove tasks
- Add subtasks by typing below a task and clicking "Add"

### Other Features
- Search tasks using the search bar
- Filter by All, Pending, Completed, or Today
- Sort by date, priority, or name
- Switch to calendar view to see tasks by date
- Check stats to see your progress
- Toggle dark/light mode with the sun/moon icon
- Get reminders for due tasks
